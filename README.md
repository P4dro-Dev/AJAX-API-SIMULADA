# AJAX-API-SIMULADA
Desenvolvimento de uma aplicação web simples para Cadastro AJAX com Tabela Dinâmica. O objetivo foi criar um formulário. JSON Server (db.json): Configuração de um arquivo db.json para simular um banco de dados. O JSON Server é usado para criar uma API RESTful localmente. Manipular dados (cadastrar, listar, editar, apagar) em uma aplicação web
